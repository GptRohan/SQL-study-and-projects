# E-commerce Sales Analysis using SQL

This project demonstrates an SQL-based analysis on a simplified e-commerce database consisting of customers, products, and orders.

## Tables Used
- Customers: Stores customer information.
- Products: Stores product details like name, category, and price.
- Orders: Tracks customer purchases and order dates.
